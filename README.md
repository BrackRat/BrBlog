# BrBlog

优雅之上的个人博客网站。

欢迎来到文艺复兴时期。

## ✨ 特征

1. 🌟 **SEO Friendly** 由 Nuxt3 驱动。
2. 🖼 **Renaissance** UI设计将带你回到 14-16 世纪
3. 📑 **Backend** 支持后台发布/编辑文章等
4. 📝 **Markdown** 使用轻量的标记语言编写文章
5. ⚡ **Lightning** 性能优秀

## 🛠️ 技术栈

- Nuxt3
- Tailwind CSS
- Prisma
- Md-editor-v3
- Pinia
- Giscus
